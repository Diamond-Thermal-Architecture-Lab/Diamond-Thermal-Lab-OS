# Validation Methods

Validation method memory is tracked in `memory/validation_methods.yml`.

## Method Selection

- Use TDTR or stack coupons to bound interface/material inputs where suitable.
- Use Raman or IR methods for temperature response under defined conditions.
- Use transient thermal testing for effective path response.
- Use cross-section and void inspection for attach quality.
- Use power cycling and incoming inspection for reliability and supplier conformance.

## Claim Safety

A validation method supports only the question it actually measures.
