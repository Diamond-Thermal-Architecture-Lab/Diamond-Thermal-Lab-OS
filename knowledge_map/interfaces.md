# Interfaces

Interface memory is tracked in `memory/tbr_ranges.yml`.

## Common Interface Questions

- Which interfaces are in the thermal path?
- Are bondline thickness, voiding, pressure, and surface finish known?
- Is interface TBR measured, supplier-stated, estimated, or unbounded?
- Could interface resistance erase a high-conductivity material benefit?

## Diamond Route Reminder

Diamond-based routes must include interface thermal resistance and bonding quality as critical uncertainties.
