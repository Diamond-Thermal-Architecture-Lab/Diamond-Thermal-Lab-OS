# Evidence Sidecars

This package provides deterministic, standard-library-only support for optional Evidence Objects, Measurement References, and Prediction-Reality Records. It stores public-safe metadata, controlled references, hashes, and traceability checks; it does not store raw data, validate a design, approve a route, or change canonical case files.

Use `scripts/labos_case.py evidence-summary <case>` for the read-only case-level view.
