## Lesson

When a GaN RF PA customer asks whether diamond is needed, the first useful output is a structured decision path, not a final architecture recommendation.

## Reusable Pattern

Start with missing thermal inputs, compare diamond and non-diamond routes neutrally, identify interface and cooling-boundary risks, and recommend bounded assumptions before detailed simulation or supplier commitment.
